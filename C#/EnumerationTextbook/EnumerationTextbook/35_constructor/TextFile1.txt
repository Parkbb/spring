생성자(Constructor)
-클래스 이름과 동일한 이름을 사용하는 메서드
- 클래스 실행 시 제일 먼저 실행되는 메서드
- 클래스(개체)를 초기화 시켜주는 메서드
	- void 키워드를 붙이지 않고 반환 값도 없음
- 생성자는 개체를 초기화
	- 클래스내의 필드를 초기화
- 현실 세계에서 자동차의 시동을 거는 동작에 비유


기본 생성자 사용
-기본 생성자으 ㅣ특징
	-public 액세스 한정자
	-클래스명과 동일한 이름 사용
	-반환값도 없고, void도 아님
	-매개변수가 없음
	-모든 필드를 zero, false 또는 null로 초기화
-생성자 문법
